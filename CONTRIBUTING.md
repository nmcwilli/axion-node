# ğŸ› ï¸ How to Contribute

1. Fork the Repository

Create your own fork of the project 

2. Clone your fork locally:

*Adjust username to whatever your account is
```
git clone https://github.com/username/axion-node
```

3. Set the main Axion Node repo as upstream

```
git remote add upstream https://github.com/nmcwilli/axion-node
```

4. Create a Branch

Create a branch in your repo for your fix or feature:
```
git checkout -b feature/my-new-feature
```

Name branches clearly:
- fix/<description>
- feature/<description>
- docs/<description>

5. Make changes and commit
```
git add .
git commit -m "Added feature X"
```

6. Write clear, maintainable code

Please follow these guidelines:
- Keep PRs focused on a single topic
- Use clear commit messages
- Maintain existing coding standards
- Add comments where logic is complex
- Include tests when applicable

7. ğŸ› Reporting Bugs

If you find a bug, please open an issue including:
- A clear title
- Steps to reproduce
- Expected behavior
- Actual behavior
- Relevant logs or screenshots

Template (if issue templates are enabled) is preferred.
